# Customer Segmentation Using RFM Analysis

## 📌 Project Overview

This project applies **RFM (Recency, Frequency, Monetary) Analysis** to segment customers based on purchasing behavior. The objective is to identify high-value customers, inactive customers, and growth opportunities to support data-driven marketing strategies.

The project uses **Python** for data preprocessing and **Power BI** for interactive dashboard development.

---

## 🎯 Objectives

- Clean and preprocess retail transaction data
- Calculate RFM metrics for each customer
- Assign RFM scores
- Create customer segments
- Build interactive Power BI dashboards
- Generate business recommendations

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI

## 📈 Customer Segments

- Champions
- Loyal Customers
- Potential Loyalists
- New Customers
- Need Attention
- At Risk
- Lost Customers

---

## 💡 Key Business Insights

- Champions generate the highest revenue.
- Loyal Customers purchase consistently.
- Lost Customers require win-back campaigns.
- Potential Loyalists can be converted into Champions.
- At Risk customers should be targeted with re-engagement campaigns.
